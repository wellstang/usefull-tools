# 唐唐工具箱 — 产品设计文档

**版本**：v0.1.0  
**更新**：2026-06-23

---

## 一、产品定位

### 一句话定位
「趁手好用的一站式效率工具集合，职场人的瑞士军刀」

### 核心价值主张
- **零门槛**：无需注册登录，打开即用
- **零隐私顾虑**：所有计算均在浏览器本地完成，不上传任何数据
- **零广告**：完全开源，GitHub Pages 托管，无商业化干扰
- **多场景覆盖**：一个域名满足产品/运营/研发/设计/日常五大场景

### 产品目标
一次性完整发布 48 个工具，所有工具开发完成后统一上线。首页只展示已完成的工具，未完成的工具不出现在页面上，不设 coming-soon 占位状态。

---

## 二、用户画像

### 主要用户群

**互联网职场人**
- 产品经理：日常需要数据计算（AB实验/NPS）、文档模板（OKR/SWOT）
- 运营同学：内容排期、UTM 链接管理、文本字数统计
- 前端/后端开发：JSON格式化、Base64、时间戳、正则测试是高频日常
- UI/UX 设计师：配色、字体、尺寸规范查询
- 通用职场场景：二维码生成、密码生成、日期计算

**核心需求特征**
- 不想安装软件，直接浏览器打开用
- 工具要快，输入即出结果，无需等待
- 对数据敏感（公司内网内容），不希望数据被上传

## 五、交互设计规范

### 首页

**布局**
- 顶部：Logo（左）+ GitHub 链接（右）
- Hero 区：工具箱名称 + 简介 + 工具总数展示
- 搜索框：居中大号，支持按名称/描述/标签模糊搜索，实时过滤
- 分类 Tab：横向排列，选中态明显，支持横向滚动（移动端）
- 工具卡片 Grid：PC 3列，平板 2列，手机 1列

**工具卡片**
- 图标（Iconify，48x48）
- 工具名称（16px 加粗）
- 简短描述（14px，最多两行）
- 分类标签（小角标）

**搜索行为**
- 输入时实时过滤，无需回车
- 搜索范围：工具名称 + 描述 + tags
- 搜索无结果时展示友好空状态提示

### 工具页

**统一结构（ToolLayout）**
```
[← 返回首页]  研发 / JSON 格式化
─────────────────────────────────
JSON 格式化
在线美化、压缩、校验 JSON 数据
─────────────────────────────────
[工具主体内容区]
```

**输入/输出规范**
- 输入区：左侧或上方，标签「输入」
- 输出区：右侧或下方，标签「输出」，右上角有「复制」按钮
- 操作按钮：主操作用蓝色实心按钮，次要操作用描边按钮
- 复制成功提示：Toast 提示 1.5 秒后消失

**错误处理**
- 输入格式错误（如 JSON 语法错误）：在输出区展示红色错误提示，不崩溃
- 空输入：按钮置灰或不执行，不报错

**响应式**
- PC（≥1024px）：左右分栏布局（输入/输出）
- 手机（<768px）：上下排列，输出在下方

## 六、UI 设计原则

### 色彩体系

| 用途 | 颜色 | Tailwind |
|------|------|----------|
| 主色 | 靛蓝 | `indigo-600` |
| 主色 Hover | 深靛蓝 | `indigo-700` |
| 背景 | 浅灰 | `gray-50` |
| 卡片背景 | 白色 | `white` |
| 边框 | 浅灰 | `gray-200` |
| 文字主色 | 深灰 | `gray-900` |
| 文字次色 | 中灰 | `gray-500` |
| 成功 | 绿 | `green-500` |
| 错误 | 红 | `red-500` |

### 圆角 & 阴影
- 卡片：`rounded-xl` + `shadow-sm hover:shadow-md`
- 按钮：`rounded-lg`
- 输入框：`rounded-lg`

### 字体
- 字体栈：`-apple-system, BlinkMacSystemFont, 'PingFang SC', 'Microsoft YaHei', sans-serif`
- 标题：600 weight
- 正文：400 weight，行高 1.6

### 间距节奏
- 卡片间距：`gap-4`（16px）
- 区块间距：`space-y-6`（24px）
- 容器最大宽度：`max-w-6xl mx-auto px-4`

### 动效
- 卡片 hover：`transition-shadow duration-200`
- Tab 切换：无动画（保持轻快）
- Toast 出现/消失：`transition-opacity duration-300`

---

## 七、移动端适配要点

- 搜索框宽度 100%
- 分类 Tab 支持横向滚动，不折行
- 工具卡片单列布局
- 输入区和输出区上下排列
- 复制按钮大小不低于 44×44px（易触点）
- 代码/文本输入框高度可拖拽（`resize-y`）
